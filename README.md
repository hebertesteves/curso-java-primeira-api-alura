# 📘 Curso Java — Criando a primeira API e conectando ao Front-end

Repositório criado para armazenar os códigos, exemplos e práticas desenvolvidas durante o curso **Java: criando a primeira API e conectando ao Front-end**, da **Alura**.

Este projeto tem como objetivo consolidar os conceitos fundamentais de **criação de APIs em Java**, integração entre **backend e frontend** e organização de aplicações utilizando boas práticas no desenvolvimento web.

---

## 🧠 Conteúdos abordados

Durante o curso, foram estudados os seguintes tópicos:

- Criação de uma **API em Java**
- Estruturação de aplicações backend
- Implementação de **endpoints REST**
- Comunicação entre **frontend e backend**
- Manipulação de requisições HTTP
- Organização do código em camadas
- Integração de dados entre aplicação e interface
- Boas práticas no desenvolvimento de APIs

---
